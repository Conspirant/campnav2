# Campus Compass

A modern, interactive campus navigation system built with React, Vite, and Tailwind CSS.

## Features

- **Interactive Map**: Detailed L-shaped floor plan with room-level precision.
- **Smart Navigation**: Smooth pathfinding animations between any two locations.
- **Location Picker**: Unified search for starting points and destinations.
- **Mobile First**: Optimized touch controls and responsive design.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```

## Deployment

This project handles client-side routing and includes a `_redirects` file for Netlify deployment.
